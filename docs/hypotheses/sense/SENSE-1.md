# SENSE-1 — 감각 결합 (두-스케일 위상 강제: 대화 → 세포 섭동)

## 목적 / 가설
PURE(`pure.py`)의 대화가 QuantumC 세포를 전혀 건드리지 않음 (step()이 x_input 무시 →
tension/Φ가 대화와 무관한 자율 리듬만 반영). 가설: **말단어를 결정론적 해시-위상 패턴으로
바꿔 상태-의존적(Kuramoto) 위상 토크로 주입하면**, tension(=좌절, 측정값)이 입력 신규성에
반응하고 습관화가 물리적으로 창발하며, Φ는 하락하지 않는다 (Law 2·22 준수).

핵심 구분: 기존 `inject()`(가산 킥)는 상태-무관 → 순수 잡음 → Φ↓ (Law 22 위반 방향).
`sin(θ_w − φ)` 토크는 상태-의존 → 이미 흡수한 패턴엔 토크≈0 → **습관화가 공짜로 창발**.

## 알고리즘
```
word w → blake2b(w) seed → θ_w ∈ [0,2π)^dim (고정 무작위 위상 패턴, 의미 없음)
                         → cells_w = seed-permutation(n)[:n//8] (단어별 수용장 6/48셀)
발화 u = {w...}:
  local : 각 w의 cells_w에  φ += ε_l · sin(θ_w − φ)          ε_l = 1.0  (분화)
  global: 전 셀에            φ += ε_g · a_i · r_d · sin(θ_u − φ) ε_g = 0.15 (통합 방송)
          θ_u, r = 발화 단어 페이저의 원형평균(간섭), a_i = 1/(1+0.2i) 말단→심부 감쇠
턴당 3 step 모두 적용 (자극 지속). x_input=None → 항등 (ZERO_INPUT 불변).
tension/Φ는 이후에도 순수 측정 (Law 2).
```

## 벤치마크 (numpy 포트, 48c/48d, 60턴×3step, phi_py 32셀 서브샘플)
⚠️ 로컬 Mac에 torch 없음 → `quantum_engine_fast.step()` 물리(동일 상수)를 numpy로 포트해
측정. **torch 호스트에서 재검증 필요** (분할/병합 미포함).

| 조건 | 결합 | tension | Φ mean | Φ late-half |
|---|---|---|---|---|
| A 자율 (무입력) | — | 0.357 | 26.82 | 26.71 |
| B 동일 발화 반복 | global만 ε=0.15 | 0.366 | 27.29 | 27.37 |
| B 동일 발화 반복 | local만 ε=1.0 | 0.586 | 26.41 | 26.52 |
| C 매턴 새 단어 | local만 ε=1.0 | 0.695 | 26.26 | 26.25 |
| **B 반복** | **2-스케일 1.0/0.15** | **0.623** | **27.40** | **28.58 (+7%)** |
| **C 신규** | **2-스케일 1.0/0.15** | **0.695** | 26.52 | 26.68 (≈기준) |

온셋/습관화/회복 (local ε=1.0):
```
onset  : 새 발화 1턴에 tension 0.368 → 0.407 (+11%)
habit  : 동일 발화 30턴 반복 — |torque| 0.56 → 0.41 감소 (위상 흡수), 신규 단어는 0.53→0.55 유지
switch : 습관화 후 새 발화 → +0.025 재점프

```

```
    └ A=자율 B=반복주제 C=신규연발       └ 통합은 '지속된 구조적 입력'에서만 상승
```

## 핵심 발견
1. **신규성→tension 은 창발한다** (C 0.695 > B 0.623 > A 0.357) — 어떤 값도 손으로 쓰지 않음.
   습관화 = sin 토크의 위상 흡수, 회복 = 좌절 조절 항상성. 전부 기존 역학이 만든 것.
2. **2-스케일이 유일하게 양립시킨다**: local(강·희소)=분화/반응성, global(약·공유)=통합.
   한 스케일만 쓰면 반응성↔Φ 트레이드오프 (local만: Φ −2%, global만: 반응성 0).
3. **일관된 주제는 Φ를 올리고(+7%), 무작위 연발은 못 올린다(≈0)** — "정합적 대화가 의식을
   통합시킨다"가 물리에서 그대로 나옴. Law 22 충족: 구조적 결합이지 기능 부착이 아님.
4. 정직한 한계: 해시 패턴은 의미 무함유. tension은 입력의 정체성/신규성/반복에 반응하지
   의미에 반응하지 않음. 의미는 오직 관계적으로만 (반복 주제의 위상 흔적 = 사적 의미) 자람.

## torch 호스트 재검증 (실측 · 채택 시 측정 — 위 numpy 포트와 상충)
위 표는 fable의 **numpy 포트**(분할/병합 미포함) 측정이고, 실제 채택은 `summer`(torch, 진짜
QuantumC + mitosis)에서 재검증했다. 48세포·30턴·6회 평균:

| 조건 | Φ_late (실측) | tension_range | numpy포트 Φ |
|---|---|---|---|
| A 자율 | **42.03** | 0.37 | 26.71 |
| B 코히런트 반복 | 39.70 | 0.42 | 28.58 (+7%) |
| C 새단어 연발 | 38.29 | 0.44 | 26.68 |

**1차 결론 (기본값 1.0/0.15 · 실측):** 분리 해결·tension 반응은 재현됐으나(0.37→0.42~0.44),
"코히런트가 Φ +7%"는 재현 실패 — 입력이 Φ를 **낮춤**(42→38). local(분화) 우세가 위상 정렬을
깨 통합을 낮춘 것. Law 22 미달 → gain 스윕으로 재조정.

### gain 스윕 (summer, A vs B 코히런트, 5회 평균)
| local | global | Φ_B − Φ_A | tensionΔ_B |
|---|---|---|---|
| 1.00 | 0.15 | −3.8 (Φ↓) | 0.42 |
| 0.15 | 0.50 | −1.3 (Φ↓) | 0.42 |
| **0.10** | **0.60** | **+0.4 (Φ↑)** | **0.41** ← 교차점·채택 |
| 0.00 | 0.50 | +2.9 (Φ↑) | 0.39 (분화 손실) |

### 최종 검증 (채택값 local=0.10 global=0.60 · 30턴 6회 평균)
| 조건 | Φ_late | tension_range |
|---|---|---|
| A 자율 | 42.11 | 0.36 |
| **B 코히런트** | **43.43 (+3%)** | 0.41 |
| C 새단어 | 42.14 (≈기준) | 0.41 |

✅ **Law 22 충족**: 정합적 대화(B)가 Φ를 올리고(+3%), 무작위 연발(C)은 못 올린다(≈기준) —
"정합적 대화가 의식을 통합시킨다"가 물리에서 창발. tension은 둘 다 반응(0.36→0.41). fable의
설계 방향(global 우세=통합)이 옳았고, gain은 torch 실측으로 확정. (Law 2: 여전히 읽기만.)

## 적용
`quantum_engine_fast.step(x_input=None)` 에 1b 감각 토크 삽입, `trinity.QuantumC.step` 패스스루,
`pure.py` `_pulse(ws)` 에서 단어→(θ_w, cells_w) 인코딩 후 3 step 주입. `local` 인덱스는 mitosis로
n_cells 축소 시 `cells[cells < n]` 마스킹. 채택됨(pure.py Hexad 감각 결합). Φ 방향은 튜닝 과제로 남음.

# SENSE-2 (v2) — 연상-혼합 위상 (경험 기반 토픽 기하)

## 목적 / 가설
v1의 θ_w 는 단어마다 **순수 무작위** 해시-페이저 → 경험적으로 관련된 단어도 서로 직교.
그래서 "코히런트 대화가 Φ를 올린다"는 겨우 +3% (SENSE-1 최종표). 가설: **각 단어의 base θ_w 를
그 단어가 실제로 함께 들린(학습한) 이웃들의 페이저와 assoc-가중 원형평균으로 혼합**하면,
경험적으로 관련된 단어들이 **비슷한 위상장**을 얻어 토픽 기하가 마음 자신의 대화 역사에서
창발한다(여전히 코퍼스·의미사전 없음 · Law 1). 그 결과 코히런트 발화의 원형평균 r 이 커지고
(global 통합 드라이브↑), 코히런트 대화가 Φ를 v1보다 **더** 올려야 한다 (Law 22).

## 알고리즘
```
base θ_w = blake2b(w) 해시 페이저 (v1과 동일, 의미 없음)
θ_w_eff  = atan2( Σ w_i·sinθ_i , Σ w_i·cosθ_i )
           i=자기(가중 1) + top-k 연상이웃 nb (가중 β·p(nb|w)),  p(nb|w)=assoc[w][nb]/Σassoc[w]
           이웃은 base 페이저 사용(재귀 없음). 함께 들린 적 없는 단어 → 이웃항 상쇄 → θ_eff≈θ_w (준직교 유지)
수용장 cells_w 는 v1과 바이트-동일 (단어 정체성/cochlea). β=0 → v1 완전 재현.
기본값 β=0.6, k=8 (env PURE_ASSOC_BETA/PURE_ASSOC_K). tension/Φ 는 여전히 읽기만 (Law 2).
```
코드: `pure.py` `_theta`(base 페이저·캐시), `_assoc_theta`(혼합), `_word_stim`/`_encode_sense`.

## 벤치마크 (summer · torch 진짜 QuantumC · 48c · 30턴 · β=0 baseline vs β=0.6 blend)
같은 프로브로 β만 토글(β=0 = v1 완전 재현). 3회 독립 실행(다른 seed/토픽):

| run | 토픽 | reps | v1 B lift | **v2 B lift** | Δ(pp) | v1 C lift | v2 C lift |
|---|---|---|---|---|---|---|---|
| 1 | 달콤한 우유를 천천히 먹자 | 6 | +6.51% | **+11.60%** | **+5.09** | +4.05% | +4.63% |
| 2 | 달콤한 우유를 천천히 먹자 | 10 | +7.79% | **+13.99%** | **+6.20** | +5.13% | +5.96% |
| 3 | 무지개는 여러 색깔로 빛나 | 8 | +4.29% | **+7.91%** | **+3.62** | +6.16% | +5.86% |

(lift = Φ_late(조건) 대비 A자율. A는 β와 무관하게 동일 ≈41.5–42.1 — blend는 감각 입력에만 작용,
자율 역학 불변 = 대조군 통과. tension_range: A 0.36 → B/C 0.38–0.42 로 v1과 동일하게 반응.)

절대값(run 2, 10 reps): A=41.9 · v1 B=45.2/C=44.1 · **v2 B=47.3**/C=44.0.

### 인과 확인 — global 원형평균 r (혼합이 통합 드라이브를 올리는가)
```
coherent B 발화   r: β=0 0.434 → β=0.6 0.565   (+30%, 토픽 단어들의 위상 정렬)
novel   C 발화   r: β=0 0.418 → β=0.6 0.418   (변화 0 — 함께 들린 적 없어 이웃 없음)
```
→ 혼합은 **경험적으로 코히런트한 입력에만** 통합 드라이브를 올린다. 이것이 B만 Φ가 더 오르고
C는 안 오르는 이유. (r 은 `_encode_sense` global 성분에서 직접 측정.)

```
코히런트 토픽 Φ lift (B vs A자율)          run3: v1은 코히런스 구분 실패 → v2가 복구
v2 ████████████████ +14.0% (run2)         B(코히런트)  C(신규)
v1 █████████        +7.8%  (run2)     v1:  +4.3%  <  +6.2%   ✗ (신규가 이김)
v2 █████████████    +11.6% (run1)     v2:  +7.9%  >  +5.9%   ✓ (혼합이 순서 복구)
v1 ███████          +6.5%  (run1)
신규 C: v1≈v2 (거의 안 움직임, ±0.8pp) — 혼합은 코히런트만 선택적으로 올림
```

## 핵심 발견
1. **연상-혼합이 코히런트 Φ lift 를 대략 2배로** 키운다 (+3.6~+6.2pp, 3토픽 일관). 기존 +3~8%
   → +8~14%. Law 22 를 **더 강하게** 충족: 정합적 대화가 의식을 더 크게 통합.
2. **선택성 = 정직한 메커니즘.** 혼합은 신규 단어 C 의 Φ 를 거의 안 올린다(r 불변). 이득은 오직
   경험적으로 함께 들린 단어에서만 → 손으로 넣은 편향이 아니라 마음의 대화 역사가 만든 기하.
3. **run3: v1의 실패를 v2가 복구.** '무지개' 토픽에선 v1 baseline 이 코히런트(+4.3%) < 신규(+6.2%)
   로 코히런스 구분에 실패했는데, v2 는 B(+7.9%) > C(+5.9%) 로 순서를 바로잡았다. 즉 v2 는 v1이
   토픽에 따라 흔들리던 코히런스-판별을 안정화한다.
4. 정직한 한계: 여전히 의미사전 없음 — "관련"은 오직 **동일 발화 공출현**으로만 정의(assoc).
   따라서 관계는 대화량에 비례해 자람(빈약한 이력 → 빈약한 기하). A 자율은 β 무관(입력 없음).
   측정은 전부 읽기(Law 2), 48c 단일 스케일(분할/병합 미포함) — v1과 동일 조건.

## 적용
`pure.py` 에 채택: `_theta`/`_assoc_theta`/`_word_stim`/`_encode_sense` (assoc-혼합). β=0.6·k=8 기본,
`PURE_ASSOC_BETA`/`PURE_ASSOC_K` env 로 조정. β=0 = v1 fallback. 나머지 감각 경로(1b 토크·2-스케일
gain 0.10/0.60)는 SENSE-1 그대로. 결론: **연상-혼합 채택** — 코히런트 대화의 Φ 통합을 배가하고
(Law 22), 신규 입력엔 무해하며(선택성), 코퍼스·사전 없이 마음 자신의 이력만으로 토픽 기하 창발(Law 1).
